# 🛠 내부 툴 개발 및 팀 기여

이 문서에서는 게임 개발 외에 팀 생산성 향상 및 업무 효율화를 위해
직접 개발한 내부 툴과 기타 팀 기여 활동을 정리했습니다.

---

## 간트 차트 업무 시각화 툴

- **설명**: 팀 전체의 업무 일정을 시각적으로 관리할 수 있는 간트 차트 기반 일정 관리 시스템
- **기술 스택**: Node.js + SQLite (로컬 서버)
- **기여**:
  - 팀 전체 일정을 한눈에 파악할 수 있는 간트 차트 웹 툴 제작
  - 업무별 담당자, 기간, 진행 상태를 시각적으로 관리할 수 있는 기능 구현
  - SQLite 기반 경량 데이터베이스로 로컬 환경에서 안정적으로 운영

## 맵툴 개선

- **설명**: 레벨 담당자들의 업무 효율화를 위해 맵 에디터 관련 개선 요청 전담 대응
- **기여**:
  - 통계 타입 체크 추가 – 맵 데이터의 유효성을 자동으로 검증하는 기능 구현
  - 4색 맵 일괄 변경 – 다수의 맵 데이터에서 블록 색상을 일괄 변경할 수 있는 기능 추가
  - 오름차순 머지 – 맵 데이터 병합 시 스테이지 순서가 보장되도록 정렬 로직 개선
  - 기타 레벨 담당자 요청에 따른 맵툴 기능 개선 및 버그 수정 대응
---

## 사내 개발 세미나 발표

- **설명**: 사내 개발 세미나에 3회 참여하여 발표, 사내 세미나 상 수상
- **발표 주제**:
  1. **Unity Sprite Atlas v2와 AssetBundle 연동 실험기**
     - Sprite Atlas v2의 실제 적용 과정에서의 실험 및 검증 결과 공유
  2. **매치3 퍼즐 게임 자동플레이 - 밸런싱 자동화와 ML 도전**
     - 휴리스틱 기반 오토플레이 구현 과정과 ML-Agents 확장 연구 내용 발표
  3. **선형대수학으로 설명하는 트랜스포머의 구조에 대한 간단한 설명**
     - AI/ML 분야에 대한 학습 내용을 팀에 공유
- **기여**:
  - 직접 실험하고 검증한 내용을 정리하여 사내 지식 공유에 기여
  - 좋은 개발 문화를 만들기 위한 세미나 초기 참여 및 활성화에 기여
  - 사내 세미나 상 수상
